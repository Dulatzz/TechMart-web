# TechMart — сайт системного интегратора безопасности

Одностраничный корпоративный сайт (landing page) для компании **TechMart** —
системного интегратора инженерной безопасности: видеонаблюдение, видеоаналитика,
СКУД и охранно-пожарная сигнализация для объектов, где важна надёжность.

Проект выполнен в рамках производственной стажировки в роли **web-стажёра**.

]

## Стек

- **React 18** — компонентный UI
- **Vite 6** — сборка и dev-сервер с hot-reload
- **CSS (vanilla)** — дизайн-токены через CSS-переменные, адаптивная вёрстка, без UI-библиотек
- Инлайновые **SVG-иконки** — без внешних иконочных пакетов

## Что реализовано

10 самостоятельных React-компонентов, каждый отвечает за свою секцию.

| Компонент | Секция |
|-----------|--------|
| `Header`    | Шапка с навигацией и CTA, sticky-позиционирование |
| `Hero`      | Первый экран: заголовок, метрики, фоновая сетка и градиент |
| `About`     | О компании + блок ключевых цифр |
| `Solutions` | Направления услуг (видеонаблюдение, видеоаналитика, СКУД, ОПС) |
| `Partners`  | Логотипы/партнёры |
| `Objects`   | Портфолио объектов по отраслям |
| `Process`   | Этапы работы с клиентом |
| `Portal`    | Описание клиентского портала + форма входа |
| `Contact`   | Контакты и форма заявки |
| `Footer`    | Подвал с навигацией и реквизитами |


## Структура проекта

```
TechMart/
├── index.html            # входная точка, монтирование React
├── vite.config.js        # конфиг Vite + плагин React
├── package.json
└── src/
    ├── main.jsx          # рендер приложения в #root
    ├── App.jsx           # композиция всех секций
    ├── styles.css        # глобальные стили и дизайн-токены
    └── components/       # 10 компонентов-секций
```

## Запуск

Требуется установленный [Node.js](https://nodejs.org/) (18+).

```bash
# установить зависимости
npm install

# режим разработки (http://localhost:5173)
npm run dev

# продакшн-сборка в dist/
npm run build

# локальный просмотр собранной версии
npm run preview
```
